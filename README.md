# Manpage

**Simple manpage template**


## Requirements

- make
- sassc


## Usage

Use `make` to build the CSS file:

``` bash
$ make style.css
```

If you don't have `sassc` installed in your prefix, you can specify the location to the executable

``` bash
$ make SASSC=path/to/sassc style.css
```
